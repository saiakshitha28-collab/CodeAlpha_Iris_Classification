# 🌸 Iris Flower Classification

## 📌 Project Overview

This project is a Machine Learning classification model developed as part of the **CodeAlpha Data Science Internship**. It predicts the species of an Iris flower based on four flower measurements using the Decision Tree Classification algorithm.

## 🎯 Objective

The objective of this project is to classify Iris flowers into one of the following three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

using machine learning techniques.

## 📂 Dataset

The project uses the Iris dataset containing:
- 150 flower samples
- 4 input features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 target classes (flower species)

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 🚀 Project Workflow

1. Load the Iris dataset
2. Explore and preprocess the data
3. Split the dataset into training and testing sets
4. Train a Decision Tree Classifier
5. Evaluate the model using accuracy and classification report
6. Visualize the data using a scatter plot
7. Predict the species of a new flower sample

## 📊 Output

- Model Accuracy
- Classification Report
- Confusion Matrix
- Iris Scatter Plot
- Sample Flower Prediction
